# Usage
<!-- usage -->
```sh-session
$ npm install -g @eaj/shipyard-cli
$ ship COMMAND
running command...
$ ship (-v|--version|version)
@eaj/shipyard-cli/0.1.0 darwin-x64 node-v12.16.1
$ ship --help [COMMAND]
USAGE
  $ ship COMMAND
...
```
